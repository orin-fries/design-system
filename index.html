user stories

As a website developer, I'd utilize a design system to provide easily scalable versions of page layouts and components, so my website can grow dynamically.


As a person with visual processing impairments, using a design system that adheres to WCAG allows me to easily use the service.


As a small business owner growing my digital presence, having a design system allows for professional-looking brand cohesion.
